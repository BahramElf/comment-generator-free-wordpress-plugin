Generate realistic comments automatically for your WordPress posts and WooCommerce products.

== Description ==
Comment Generator helps you populate your WordPress site with realistic-looking comments. Perfect for development, testing, and demonstration purposes.

Key Features:

* Generate comments for posts and WooCommerce products
* Customize comment sentences for different scenarios
* Control comment frequency and timing
* Manage buyer and non-buyer comments for products
* Set comment status (approved/pending)
* Configure product stock status impact

== Installation ==
1. Upload the `comment-generator` to `/wp-content/plugins/comment-generator` directory
2. Activate the plugin through the 'Plugins' screen in WordPress
3. Use the Settings->Comment Generator screen to configure the plugin

== Frequently Asked Questions ==
= Can I delete generated comments? =
Yes, you can easily remove all generated comments using the "Delete Commented Items" button in the plugin settings.

= Does it work with WooCommerce? =
Yes, the plugin fully supports WooCommerce product comments with buyer/non-buyer variations.
